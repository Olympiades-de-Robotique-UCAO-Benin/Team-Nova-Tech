Robot suiveur de ligne avec évitement d'obstacles.

Le projet Nova Tech vise à concevoir un robot autonome performant, précis et réactif, capable de suivre une ligne noire et de contourner les obstacles pour remporter le concours UCAO TECH.

But du projet

Objectifs et idéaux recherchés.

Le robot est construit pour la compétition, avec un focus sur le mouvement dynamique, la réactivité et la précision.

🖤

Suivi de ligne

Les capteurs infrarouges détectent la ligne noire et envoient les données à l'Arduino qui corrige la direction en continu.

Évitement d'obstacles

Le robot détecte les obstacles sur son parcours puis adapte sa trajectoire pour les contourner sans perdre le suivi.

Mission 2
🎯

Principe de fonctionnement

Du capteur au mouvement.

Le système suit une chaîne simple : capteurs → microcontrôleur → interface de puissance → moteurs.

Chaîne de traitement

Lecture ligne noire, décision de correction, commande moteur, puis ajustement de trajectoire en boucle.

// Logique simplifiée Arduino int g = readIR(LEFT); int c = readIR(CENTER); int d = readIR(RIGHT); if(obstacleDetected()){ avoidObstacle(); } else { followLine(g,c,d); }

Cadrage projet

Domaines, tâches et matériel.

Synthèse du cahier de charge : application concours, technologie et ressources.

01 — Domaine

Application visée

  • Utilisation dans le cadre d'un concours de robotique
  • Parcours à ligne noire avec obstacles
  • Objectif : performance, précision, réactivité
02 — Technologie

Base technique

  • Microcontrôleur principal : Arduino UNO
  • Programme embarqué pour la prise de décision
  • Interface de puissance pour piloter les moteurs
03 — Matériel

Liste principale

  • Arduino UNO, driver moteur, 2 moteurs DC
  • 3 capteurs IR, 3 roues, châssis, breadboard
  • Fils jumper, interrupteur, piles/batteries
04 — Tâches

Travaux à réaliser

  • Conception du robot
  • Développement du programme
  • Assemblage matériel
  • Tests, optimisation, préparation concours
