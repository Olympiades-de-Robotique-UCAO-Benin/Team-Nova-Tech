Logiciel embarqué

Structure du programme Arduino

Organisation simple et efficace du code Arduino pour le robot suiveur de ligne avec obstacle avoidance.

setup() Initialisation

Configuration initiale : initialiser les broches (INPUT/OUTPUT), configurer la communication série (9600 baud), charger les seuils calibrés.

readSensors() Capteurs

Lire les 3 capteurs IR (A0, A1, A2) et le module obstacle (D2). Comparer avec seuils pour générer détections booléennes.

followLine() Logique

Table de vérité sur les 3 capteurs. Sélectionner vitesses moteur (gauche/droite) selon la configuration détectée.

avoidObstacle() Sécurité

Si obstacle détecté → arrêter ou contourner. Priorité sur le suivi de ligne pour la sécurité mécanique.

setMotors() Actuation

Contrôler les 2 moteurs DC via PWM (pins 5, 10) et direction (pins 8, 9, 11, 12). Convertir vitesses (0-255) en signaux.

loop() Boucle

Répétée continuellement (~100 Hz) : lire capteurs → décider action → commander moteurs → enregistrer événements.

Flux d'exécution

loop() {
  // 1. Lire les capteurs IR
  readSensors();

  // 2. Vérifier obstacle (priorité)
  if (obstacleDetected) {
    avoidObstacle();
    return;
  }

  // 3. Suivre la ligne avec les 3 capteurs
  followLine();

  // 4. Enregistrer état pour diagnostic
  logSensorState();
  
  // ~10ms entre cycles
  delay(10);
}
